# 🚚 TransitOps - Smart Transport Operations Platform

## Overview
TransitOps is a Smart Transport Operations Platform developed for a Hackathon. It helps organizations manage vehicles, drivers, trips, maintenance, fuel expenses, and fleet analytics.

## Features

- Secure Login & Role-Based Access Control
- Vehicle Registry
- Driver Management
- Trip Management
- Maintenance Workflow
- Fuel & Expense Tracking
- Dashboard Analytics

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Spring Boot
- Spring Security
- MySQL
